Politique de Confidentialité

Polyglot Subtitle Translator — Extension Chrome

1. Introduction

Polyglot Subtitle Translator (ci-après « l'Extension ») est une extension de navigateur Chrome qui traduit les sous-titres de vidéos en temps réel. Cette politique de confidentialité décrit comment l'Extension traite vos données.

2. Données collectées

Aucune donnée personnelle n'est collectée.

L'Extension ne collecte, ne stocke et ne transmet aucune information personnelle identifiable, notamment :

  • Aucun nom, adresse e-mail ou identifiant
  • Aucun historique de navigation
  • Aucune donnée de géolocalisation
  • Aucun cookie de suivi

3. Données stockées localement

L'Extension utilise chrome.storage.sync pour enregistrer uniquement vos préférences :

  • Liste des sites configurés (nom de domaine, langues source et cible)
  • Préférences d'affichage (taille des sous-titres, mode de traduction)
  • Langues par défaut

Ces données sont stockées localement sur votre navigateur et synchronisées via votre compte Google Chrome si la synchronisation est activée. Elles ne sont accessibles que par vous.

4. Services tiers

L'Extension utilise l'API MyMemory Translation pour effectuer les traductions. Lors d'une traduction :

  • Le texte des sous-titres est envoyé à l'API MyMemory pour traduction
  • Aucune information personnelle n'est transmise avec la requête
  • Consultez la politique de MyMemory pour plus de détails

5. Permission « Tous les sites »

L'Extension demande la permission d'accéder à tous les sites web (<all_urls>) car elle doit pouvoir détecter et traduire les sous-titres sur n'importe quel site de vidéo que l'utilisateur configure. L'Extension n'est active que sur les sites explicitement ajoutés par l'utilisateur dans ses paramètres.
